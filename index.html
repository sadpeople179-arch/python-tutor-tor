📚 Pendahuluan

Python adalah bahasa pemrograman yang mudah dipelajari dan sangat powerful. Dirancang dengan filosofi "code readability counts", Python membuat kode Anda lebih mudah dibaca dan dimengerti.

Mengapa Belajar Python?

  • Mudah Dipelajari: Sintaks yang sederhana dan intuitif
  • Versatile: Digunakan untuk web development, data science, AI, automation, dan banyak lagi
  • Popular: Komunitas yang besar dan resources yang melimpah
  • High Demand: Banyak perusahaan mencari developer Python

Instalasi Python

Kunjungi python.org dan download versi terbaru. Pastikan untuk mencentang "Add Python to PATH" saat instalasi.

💡 Tips: Verifikasi instalasi dengan membuka Command Prompt/Terminal dan ketik: python --version

🎯 Variabel dan Assignment

Variabel adalah wadah untuk menyimpan nilai data. Dalam Python, Anda tidak perlu mendeklarasikan tipe data karena Python akan mendeteksi tipenya secara otomatis.

Membuat Variabel

# Syntax: nama_variabel = nilai
nama = "Budi"
umur = 25
tinggi = 175.5
aktif = True

# Menampilkan nilai variabel
print(nama)
print(umur)

Naming Convention

  • Mulai dengan huruf atau underscore (_)
  • Gunakan lowercase untuk nama variabel (snake_case)
  • Hindari spasi, gunakan underscore untuk pemisah kata
  • Hindari kata kunci Python sebagai nama variabel
✓ Contoh yang Benar:
nama_lengkap = "Budi Santoso"
tahun_lahir = 1999

📦 Tipe Data

Python memiliki beberapa tipe data bawaan yang fundamental untuk programming.

Tipe Data Utama

Tipe Data Deskripsi Contoh
str String (teks) "Hello World"
int Integer (bilangan bulat) 42
float Float (bilangan desimal) 3.14
bool Boolean (True/False) True
list List (array) [1, 2, 3]
dict Dictionary (key-value) {"nama": "Budi"}

Contoh Penggunaan

# String
nama = "Andi"

# Integer
nilai = 85

# Float
gpa = 3.75

# Boolean
lulus = True

# List
buah = ["apel", "jeruk", "mangga"]

# Dictionary
siswa = {"nama": "Budi", "umur": 20}

Mengecek Tipe Data

print(type(nama)) # <class 'str'>
print(type(nilai)) # <class 'int'>
print(type(buah)) # <class 'list'>

⚙️ Operator

Operator digunakan untuk melakukan operasi pada variabel dan nilai.

Operator Aritmetika

a = 10
b = 3

print(a + b) # Penjumlahan: 13
print(a - b) # Pengurangan: 7
print(a * b) # Perkalian: 30
print(a / b) # Pembagian: 3.333...
print(a // b) # Pembagian bulat: 3
print(a % b) # Modulo (sisa): 1
print(a ** b) # Pangkat: 1000

Operator Perbandingan

a = 10
b = 5

print(a == b) # Sama dengan: False
print(a != b) # Tidak sama dengan: True
print(a > b) # Lebih dari: True
print(a < b) # Kurang dari: False
print(a >= b) # Lebih dari atau sama: True
print(a <= b) # Kurang dari atau sama: False

Operator Logika

a = True
b = False

print(a and b) # AND: False
print(a or b) # OR: True
print(not a) # NOT: False

🔀 Control Flow

Control flow memungkinkan Anda membuat keputusan dalam program dan mengulangi tindakan tertentu.

If-Elif-Else

nilai = 85

if nilai >= 80:
    print("Grade A")
elif nilai >= 70:
    print("Grade B")
else:
    print("Grade C")

Loop - For

# Iterasi list
buah = ["apel", "jeruk", "mangga"]
for b in buah:
    print(b)

# Dengan range
for i in range(5):
    print(i) # 0, 1, 2, 3, 4

Loop - While

i = 0
while i < 5:
    print(i)
    i += 1

🔧 Fungsi

Fungsi adalah blok kode yang dapat digunakan kembali. Membuat fungsi membantu mengorganisir kode Anda.

Membuat Fungsi Sederhana

def sapa(nama):
    print("Halo, " + nama)

sapa("Budi") # Output: Halo, Budi

Fungsi dengan Return

def tambah(a, b):
    return a + b

hasil = tambah(10, 5)
print(hasil) # Output: 15

Fungsi dengan Default Parameter

def greet(nama, greeting="Halo"):
    return f"{greeting}, {nama}!"

print(greet("Andi")) # Halo, Andi!
print(greet("Andi", "Selamat pagi")) # Selamat pagi, Andi!

    if operasi == '+':
        return a + b
    elif operasi == '-':
        return a - b
    elif operasi == '*':
        return a * b
    elif operasi == '/':
        return a / b if b != 0 else "Tidak bisa dibagi nol"

print(kalkulator(10, 5, '+')) # 15
